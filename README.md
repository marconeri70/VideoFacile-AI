# VideoFacile AI

Prima versione gratuita di una PWA per creare progetti video social.

## Cosa fa
- Genera copione parlato
- Genera storyboard
- Genera prompt per immagini/video AI
- Genera prompt copertina
- Genera descrizione social e hashtag
- Salva i progetti nel browser
- Funziona come PWA installabile

## Come pubblicarla su GitHub Pages
1. Crea un nuovo repository GitHub.
2. Carica tutti i file contenuti in questa cartella.
3. Vai su Settings > Pages.
4. Scegli Deploy from a branch.
5. Seleziona branch main e cartella /root.
6. Apri il link pubblicato da GitHub Pages.

## Nota
Questa versione non usa API esterne e non genera video finali da sola.
Serve per preparare tutto il materiale da usare in CapCut, Canva, Gemini, Pika, Runway, Bing Video Creator o altri strumenti.
